고객센터 화면 기능 : 기능 구현 화면들의 메뉴바 구성, 메뉴바를 통한 기능 구현 화면들로 이동 기능 구현,
메인 화면은 자주 찾는 질문 OR 해당 회원의 문의 내역 테이블을 제공하는 기능 구현

자주 찾는 질문 : 자주 찾는 질문 게시판 구현, 회원의 게시글 열람가능 기능 구현

공지게시판 : 공지 게시판 구현, 공지 게시글 열람가능 기능 구현.

이메일 문의 : 문의별 담당 관리자의 이메일 정보와 이메일 문의의 작성 양식 정보를 제공하는 페이지 구현

내 상담내역 : 해당 회원의 회원가입 일자부터 현재까지 문의한 내역의 열람 가능한 페이지 구현, 날짜별 검색 기능 구현, 
대분류로 나뉘며 '답변완료', '미답변' 분류로 나뉨.

질문 작성 : 1:1 문의에 대한 글 작성이 가능한 페이지 구성, 수정 및 삭제는 불가능.


마이페이지 화면 기능 : DB에 저장된 로그인 한 해당 회원의 정보를 제공,
메뉴를 통한 기능 페이지들로의 이동기능 구현

내정보 기능 : DB에 저장된 해당 회원의 정보를 제공 및 수정이 가능한 기능 구현

이벤트 참여내역 : 해당 회원의 참여중인 이벤트 내역을 제공 및 회원이 당첨된 이벤트 내역 제공

포인트 사용내역 : 해당 회원의 보유중인 포인트 정보를 제공 및 이전 사용했던 포인트와 기간 정보를 제공

쿠폰 : 회원이 보유 중인 쿠폰 수량, 쿠폰 정보를 제공, 사용했던 쿠폰의 기간과 사용처, 사용한 쿠폰정보를 제공

예매내역 : 회원이 최근 1개월간 예매한 정보를 제공, 예매한 영화의 제목, 주소, 관, 좌석, 상영날짜, 시간 정보를 제공, 
영화 제목 클릭 시 영화 상세페이지로 이동기능구현, 예매취소 기능을 구현

내가 본 영화 : 회원이 가입 후 현재까지 관람한 영화들의 정보를 제공, 평점 쓰기 페이지로의 이동 기능 구현

내가 쓴 평점 : 회원이 가입 후 현재까지 관람한 영화들에 작성한 평점들의 정보를 제공, 원하는 평점으로의 페이지 이동 구현

나의 문의 : 1:1 고객센터 페이지에 회원이 작성하였던 문의 정보를 제공, 1:1문의 기능의 구현, 문의에 대한 답변이 달릴 시 알림기능 구현

나의 등급 : 현재 회원의 등급 정보와 이전 등급의 정보, 승급 날짜, 등급혜택안내 정보를 제공하는 기능 구현

내 스토어 샵 : 현재 회원이 보유(구매한)상품 품목 리스트 내역을 제공, 상품의 정보 클릭 시 상품 상세페이지로의 이동 기능 구현, 
상품의 이름, 유효기간, 가격, 상품사진, 구매날짜등 상품의 정보를 제공하는 기능 구현, 상품에 대한 환불기능 구현

프로필 관리 : 회원의 프로필 사진, 닉네임 정보를 제공 및 해당 정보를 수정 할 수 있는 기능 구현.
