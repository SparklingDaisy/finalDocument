관리자 마이페이지 : 로그인 한 관리자의 직책, 이름등 DB에 저장된 관리자의 정보를 제공하는 기능 구현, 
관리 기능 페이지들로의 이동 기능 구현

영화관 관리 : '영화관 등록관리'(담당 : 용상민) 기능을 구현한 페이지로의 이동 기능을 구현
스토어 관리 : '관리지 스토어'(담당 : 김갈매) 기능을 구현한 페이지로의 이동 기능을 구현
매출 내역 : '관리자 매출 관리'(담당 : 김한민) 기능을 구현한 페이지로의 이동 기능을 구현
회원 관리 : '관리자 회원 관리'(담당 : 김한민) 기능을 구현한 페이지로의 이동 기능을 구현
평점 관리 : '관리자 평점 관리'(담당 : 조선영)기능을 구현한 페이지로의 이동 기능을 구현
예매 현황 : '관리자 예매 현황'(담당 : 조선영) 기능을 구현한 페이지로의 이동 기능을 구현
영화 관리 : '관리자 영화 관리'(담당 : 조선영) 기능을 구현한 페이지로의 이동 기능을 구현
이벤트 관리 : '관리자 이벤트 관리'(담당 : 조선영) 기능을 구현한 페이지로의 이동 기능을 구현


고객센터 : 
'관리자 고객센터' 페이지로의 이동, 회원들이 문의한 문의 내용을 테이블 타입으로 열람 가능,
테이블의 대분류는 '상담완료문의' 와 '미상담문의' 로 분류, 미상담 문의의 게시글 선택 시 답변 기능 구현,
자주 찾는 질문 게시판 구현 및 글 등록, 수정, 삭제 기능 구현, 공지게시판 구현 및 글 등록, 수정, 삭제 기능구현,
이메일 문의에 대한 관리자만 열람 가능한 저장용 게시판 구현, 내 상담내역 게시판 구현,
내 상담내역 게시판에선 해당 관리자가 답변해준 문의 내역 열람가능 기능 구현,
헤드 관리자는 모든 관리자가 답변한 문의 내역의 열람 가능 기능을 구현.
